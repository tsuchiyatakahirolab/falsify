# Demo Scenarios

These are scenario templates, not pre-judged conclusions.

## Scenario A — Strategic narrative about Japan

Use a current, public, official source containing a claim about Japan's defense/security posture.

Demonstrate:
- original statement and source;
- atomic factual and inferential claims;
- what is supported;
- what is disputed;
- historical analogy or attribution if present;
- contradictory/qualifying evidence;
- consistency test where relevant;
- challenge to Falsify's own rebuttal.

The final demo source should be manually reviewed before recording.

## Scenario B — Academic citation mismatch

Use a synthetic or openly shareable research passage where:
- citation exists;
- source is relevant;
- specific magnitude or causal claim is not supported.

Demonstrate why "real citation" is not the same as "citation supports this sentence."

## Scenario C — Time-sensitive public claim

Use a claim with "currently", "largest", "latest", or equivalent language tied to an old source.

Demonstrate temporal audit.

## Demo selection rule

Scenario A is the flagship. Scenario B or C should be available as a neutral fallback to show generality.
