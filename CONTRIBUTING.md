# Contributing to Falsify

Thank you for helping improve Falsify.

Before contributing, read:
- `README.md`
- `docs/PROJECT_CHARTER.md`
- `docs/SECURITY_PRIVACY.md`
- `docs/OPEN_SOURCE_GOVERNANCE.md`

Contributions should improve inspectability, evidence provenance, adversarial testing, accessibility, security, or evaluation.

Please include tests or reproducible examples for behavior changes.

Do not submit confidential documents, proprietary datasets, or personal data in issues or fixtures.

For incorrect findings, report:
1. the claim;
2. the Falsify verdict;
3. the disputed evidence or reasoning;
4. a source supporting the correction;
5. whether the problem is claim decomposition, evidence retrieval, source fit, inference, or another class.
