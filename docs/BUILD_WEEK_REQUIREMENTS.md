# OpenAI Build Week Requirements Snapshot

Verified against official OpenAI Build Week / Devpost pages on 2026-07-17.

## Core requirement

Build a working project using **Codex with GPT-5.6**.

## Tracks

- Apps for Your Life
- Work & Productivity
- Developer Tools
- Education

Current intended Falsify track: **Work & Productivity**.

## Submission items

Official guidance lists:
- working project;
- category;
- project description;
- public YouTube demo video under 3 minutes;
- demo voiceover explaining what was built and how both Codex and GPT-5.6 were used;
- code repository;
- public repo with relevant open-source license;
- README with setup instructions and sample data if needed;
- README highlighting Codex acceleration, key decisions, and GPT-5.6/Codex use;
- `/feedback` Codex Session ID for the session where the majority of core functionality was built;
- extra install/platform/test guidance for plugins or developer tools.

- New repository for Falsify.

- Primary Codex build thread retained through the core build.
